# What is Project Lips
Lips is a project for changing lips color or installing virtual lipstick on your lips

# Requirements
1.npm\
to install npm run `apt-get install npm nodejs`\
3.django\
to install django run `pip3 install django` or `pip install django`\
5.python3\
to install python run `apt-get install python`


# How to setup project
1.first of all clone the directory using `git clone` \
2.goto the `/lips` directory then open terminal in current folder\
3.run command `npm install` this command will install node modules for reactjs.\
4.run `npm start` command\
5.goto `/backend` folder then run command `python3 manage.py runserver` \
